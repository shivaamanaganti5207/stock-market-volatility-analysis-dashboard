# Stock Market Volatility Analysis Dashboard

## Overview

This project analyzes stock market price volatility using candlestick charts, return distribution analysis, and interactive visualizations.

The project was developed as a Real-Time Research Project (RTRP) using Python, Pandas, NumPy, and Plotly.

## Features

- Candlestick Chart Analysis
- Return Distribution Visualization
- Volatility Analysis
- Correlation Heatmap
- RSI Indicator Analysis
- MACD Indicator Analysis
- Interactive Dashboard

## Technologies Used

- Python
- Pandas
- NumPy
- Plotly
- Jupyter Notebook

## Dataset

The dataset contains more than 13,000 stock market records including:

- Date
- Open
- High
- Low
- Close
- Volume
- RSI
- MACD
- Sentiment
- Target

## Project Outcomes

- Identified stock price volatility patterns
- Visualized bullish and bearish trends
- Analyzed return distributions
- Studied relationships between market indicators
